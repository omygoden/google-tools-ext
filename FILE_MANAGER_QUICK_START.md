# 文件管理模块 - 快速开始指南

## 🚀 快速开始

### 1. 打开文件管理模块

1. 打开 Chrome 扩展
2. 点击顶部的 **📁 文件管理** 标签

### 2. 上传文件

**方式一：点击选择**
- 点击 **📂 选择文件** 按钮
- 在弹出的文件选择器中选择一个或多个文件
- 支持的格式：`.doc`, `.docx`, `.txt`, `.md`, `.markdown`, `.html`, `.htm`

**方式二：拖拽上传**
- 直接将文件拖拽到上传区域
- 松开鼠标即可完成上传

### 3. 查看文件列表

上传后，文件会显示在列表中，每个文件显示：
- 📘 `.docx` - Word 2007+ 文档
- 📗 `.doc` - Word 97-2003 文档
- 📝 `.txt` - 纯文本文件
- 📋 `.md` - Markdown 文档
- 🌐 `.html` - HTML 文件
- 文件名
- 文件大小（KB）
- 移除按钮

### 4. 设置分隔符（可选）

在"文件间分隔符"区域选择：
- **单行空白**（默认）- 文件之间一个空行
- **双行空白** - 文件之间两个空行
- **分隔线** - 60 个"-"字符
- **星号分隔** - 60 个"*"字符
- **自定义** - 输入自定义分隔符（支持 `\n` 换行符）

### 5. 合并文件

**合并为 TXT**
- 点击 **合并为 TXT** 按钮
- 自动下载 `merged_documents.txt` 文件
- 预览区域显示合并后的内容

**合并为 Word**
- 点击 **合并为 Word** 按钮
- 自动下载 `merged_documents.docx` 文件
- 可用 Word 或其他编辑器打开

### 6. 管理文件

**移除单个文件**
- 点击文件右侧的 **移除** 按钮

**清空所有文件**
- 点击 **清空列表** 按钮
- 所有文件和预览内容将被清空

## 📝 使用示例

### 示例 1：合并多个 Markdown 笔记

1. 准备 3 个 Markdown 文件：
   - `note1.md` - 项目概述
   - `note2.md` - 技术方案
   - `note3.md` - 实施计划

2. 上传所有文件到文件管理模块

3. 选择"分隔线"作为分隔符

4. 点击"合并为 TXT"

5. 得到一个包含所有笔记的完整文档

### 示例 2：整理 Word 文档

1. 准备多个 Word 文档章节

2. 上传所有 `.docx` 文件

3. 选择"双行空白"作为分隔符

4. 点击"合并为 Word"

5. 得到一个合并后的 Word 文档

### 示例 3：合并 HTML 文档

1. 准备多个 HTML 文件

2. 上传所有 `.html` 文件

3. 选择合适的分隔符

4. 点击"合并为 TXT"

5. 得到包含所有 HTML 代码的文本文件

## 💡 使用技巧

1. **混合文件类型**
   - 可以同时上传不同类型的文件
   - 例如：`.docx` + `.txt` + `.md`

2. **自定义分隔符**
   - 使用 `\n` 表示换行
   - 例如：`\n=== 第 X 章 ===\n`

3. **预览功能**
   - 合并后内容会显示在预览区域
   - 可以在下载前检查内容

4. **批量处理**
   - 一次可以上传多个文件
   - 按上传顺序合并

5. **文件编码**
   - 建议使用 UTF-8 编码
   - 确保中文正确显示

## ⚠️ 注意事项

1. **Word 文档格式**
   - 合并时会丢失原始格式（字体、颜色等）
   - 只保留纯文本内容

2. **HTML 文件**
   - 保留原始 HTML 标签
   - 不会渲染为可视化内容

3. **文件大小**
   - 建议单个文件不超过 10MB
   - 避免浏览器内存溢出

4. **`.doc` 文件**
   - 建议转换为 `.docx` 格式
   - 以获得更好的解析支持

## 🔧 故障排除

### 问题：文件无法上传

**解决方案：**
- 检查文件格式是否支持
- 确认文件没有损坏
- 尝试重新选择文件

### 问题：中文显示乱码

**解决方案：**
- 确保源文件使用 UTF-8 编码
- 使用文本编辑器转换编码
- 重新保存文件

### 问题：合并后内容不完整

**解决方案：**
- 检查源文件是否完整
- 确认文件格式正确
- 查看浏览器控制台错误信息

### 问题：Word 文档无法解析

**解决方案：**
- 将 `.doc` 转换为 `.docx`
- 使用 Word 另存为新格式
- 检查文件是否加密
