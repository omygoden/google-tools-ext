# 文件管理模块 - 自定义分隔符功能说明

## 🎉 新功能：自定义文件分隔符

现在您可以自由选择文件之间的分隔方式，让合并后的文档更符合您的需求！

## 📋 分隔符选项

### 1. 单行空白（默认）✨
**效果**：
```
文件1内容

文件2内容
```

**适用场景**：
- 简洁的文档合并
- 内容紧凑排列
- 默认推荐选项

---

### 2. 双行空白
**效果**：
```
文件1内容


文件2内容
```

**适用场景**：
- 需要更明显的视觉分隔
- 文档章节区分
- 提高可读性

---

### 3. 分隔线
**效果**：
```
文件1内容

------------------------------------------------------------

文件2内容
```

**适用场景**：
- 正式文档
- 需要明确分隔的内容
- 报告、总结类文档

---

### 4. 星号分隔
**效果**：
```
文件1内容

************************************************************

文件2内容
```

**适用场景**：
- 装饰性分隔
- 创意文档
- 突出显示分隔

---

### 5. 自定义分隔符 🎨
**功能**：输入任何您想要的分隔内容

**示例 1 - 简单文本**：
```
输入：===
效果：
文件1内容
===
文件2内容
```

**示例 2 - 多行分隔**：
```
输入：\n--- 分隔 ---\n
效果：
文件1内容

--- 分隔 ---

文件2内容
```

**示例 3 - 特殊符号**：
```
输入：\n▼▼▼\n
效果：
文件1内容

▼▼▼

文件2内容
```

**示例 4 - 章节标记**：
```
输入：\n\n【下一文档】\n\n
效果：
文件1内容


【下一文档】


文件2内容
```

## 🔧 转义字符支持

在自定义分隔符中，您可以使用以下转义字符：

| 转义字符 | 含义 | 示例 |
|---------|------|------|
| `\n` | 换行符 | `\n\n` = 两个换行 |
| `\t` | 制表符（Tab） | `\t---\t` = Tab + --- + Tab |
| `\r` | 回车符 | `\r\n` = Windows 换行 |

**使用示例**：
```
输入：\n\n========\n\n
实际效果：

========

```

## 🎯 实际应用场景

### 场景 1：合并会议记录
**选择**：分隔线
**原因**：清晰区分不同会议的内容

### 场景 2：合并日记
**选择**：自定义 `\n\n【日期分隔】\n\n`
**原因**：明确标识不同日期

### 场景 3：合并报告
**选择**：双行空白
**原因**：保持专业性，适度分隔

### 场景 4：合并笔记
**选择**：单行空白
**原因**：紧凑排列，节省空间

### 场景 5：合并文章
**选择**：自定义 `\n\n* * *\n\n`
**原因**：文学作品常用的分隔方式

## 📝 操作步骤

1. **上传文件**
   - 选择或拖拽多个 Word 文档

2. **选择分隔符**
   - 点击分隔符选项中的单选按钮
   - 如果选择"自定义"，在输入框中输入分隔内容

3. **合并导出**
   - 点击"合并为 TXT"或"合并为 Word"
   - 查看预览区域确认效果

4. **调整优化**
   - 如果效果不满意，可以重新选择分隔符
   - 再次点击合并按钮

## ⚠️ 注意事项

### 1. 分隔符长度
- 建议不超过 100 个字符
- 过长的分隔符可能影响阅读体验

### 2. 特殊字符
- 某些特殊字符在 Word 中可能显示异常
- 建议使用常见的 Unicode 字符

### 3. 转义字符
- 在自定义输入框中，`\n` 会被转换为真正的换行
- 如果想显示 `\n` 文本，请使用 `\\n`

### 4. 预览确认
- 合并后先查看预览区域
- 确认分隔效果符合预期
- 再下载文件

## 💬 常见问题

### Q1：自定义分隔符不生效？
**A**：请确保：
1. 选中了"自定义"单选按钮
2. 在输入框中输入了内容
3. 重新点击了合并按钮

### Q2：如何输入多行分隔符？
**A**：使用 `\n` 转义字符，例如：`\n\n分隔\n\n`

### Q3：分隔符在 Word 中显示不正常？
**A**：
- 某些特殊字符可能不被 Word 支持
- 建议使用常见字符和符号
- 可以先导出为 TXT 查看效果

### Q4：能否为不同文件使用不同分隔符？
**A**：当前版本所有文件使用相同分隔符。如需不同分隔符，建议分批合并。
