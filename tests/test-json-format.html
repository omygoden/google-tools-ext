📝 使用说明

  1. 打开Chrome扩展的app.html页面
  2. 切换到"JSON 格式化"标签
  3. 复制上面的测试用例到输入框中
  4. 观察输出框的变化：
    • 大数据量：应该看到加载动画
    • 错误JSON：应该在输出框看到红色的错误信息
    • 正常JSON：应该正常格式化，文字为黑色
